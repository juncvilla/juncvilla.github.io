This is how to make tables in HTML

Table tags

Tag Name Description
<table> Table Wraps all table elements
<thead> Table head Sections off the header row of a table
<tbody> Table body Sections off the content of a table
<tr> Table row Defines a new row of (horizontal) table data in a table
<td> Table data Contains the displayed content in a table
<tfoot> Table footer Sections off the footer in a table

Table attributes

Tag Name Description
colspan Column span Defines how many columns a td should span across (horizontal)
rowspan Row span Defines how many rows a td should span across (vertically)
